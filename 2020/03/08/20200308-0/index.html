雨季

发表于 2020-03-08 更新于 2022-02-06 分类于 思维模式 本文字数： 337 阅读时长 ≈ 1 分钟

灰色在天空中晕染开来，暮色四合

雨连着下了一天，我提着手机，歪坐在阳台上。时而看看综艺，时而看看春雨。

似乎是为了吸引我的目光，雨来势渐猛，淅淅沥沥到哗哗啦啦。透过阳台，能看见满树的玉兰像雪花一样滴落，在雨中打几个转，不甘的撞击大地。突然想带着相机下去，记录玉兰花瓣的雨中盛景。

没带雨伞便匆匆出门，刚刚停歇的雨又开始唱歌。

无奈之中，只能在屋檐下望雨。

灰色在天空中晕染开来，暮色四合。咫尺的白玉兰被雨隔断，而我只能望花兴叹。人群来往匆匆，戴着口罩，提着干粮，埋头向前，只有我穿着臃肿的睡衣，站在檐下看雨和花。

收起相机，一咬牙便冲进雨幕，跳跃着躲过水洼，抬眼，便是花雨的世界。

雨水激起的雾气笼罩在玉兰树中，满地，满眼，满树的白花点缀了灰色的傍晚。
