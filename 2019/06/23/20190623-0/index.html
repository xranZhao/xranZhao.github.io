    复盘

    拿状态最好的一周 20190519-20190525 来与这周比较，状态真是差的不是一星半点。

    对于这一周的印象就停留在最倒霉的一周、睡不够、来往于漫咖啡和人际交往。好像所有的琐事都集中在了这一周，本来想通过减个头发，来去去霉气，可是竟然越来越倒霉，都不知道要怎么披着短发出门，好难过呀。数据挖掘的实验报告还堆着等我来写，下一周还有4门考试，可是自己都还没有开始复习。还有专业课上的内容，自己都还没有开始学习。我到底在干什么呀！每次找教室的时候，教室就被占有来考试，我要烦死了。

    自己原来的学习状态似乎也没有了，就像是懈怠了一样，感觉坐不住一样。曾经我通过读书获取的精神力量，似乎都忘记了。六个月实在是太长了，我似乎忘记了好多东西。这个文章写完以后，就要去季度复盘了。
