总结

已经不算是第一周开始了，所有的计划都被打乱了。

因为一个故事。

我猜想是六个月的时间期限已到，所有六个月前的满心欢喜和一腔热血都会随着时间逐渐消失。六个月应该是我第一次意识到的时间，是我记忆的极限。最近半月发生的事情不可谓不多，感冒一个星期，然后逍遥的逃课一个星期，我的时间花销上竟然没有一天用于中心工作。写下这段话，有些艰难，就像是一点点撕开自己的虚伪和懦弱，让自己面对。可是有什么办法呢？事情发生了，就不会回去。能做的，不过是从头再来。

所以，我又要开始一个记忆周期的轮回，至于这一次的轮回会持续多久，不知道。

近些天一直在看同人小说，看犬夜叉中的桔梗，看红楼中的黛玉。这是我以前万万想不到会于我有联系的两个人，而我竟然会拿出一周的时间，不断的翻找那些与他们相关的故事。好奇怪，感觉自己的感情突然在这一周蓬勃而出，就像是一点都不想要克制一样，我就随着自己的感情起伏，想干什么就干什么，可是自己做的最多的事情，不过是看书看书，在床上看书。

与别人住在一起的好处，便是可以缩减自己的放纵时间，即使自己是在放纵，也会有羞愧之心。于是，在昨天的一顿火锅之后，wisdom 终于回来了。然后今天便又回到了这个教室，这个自己待了3个月的教室。

事情还是很多，多到我不知道终点在哪里。苦涩还是有的，但是更多的是庆幸，庆幸自己总算是又扛过了一次挑战。即使是放纵的时候，自己的内心也知道，说，这只是一次情绪的释放，释放过后便是再一次的理智回归。因为我不喜欢这种被情绪掌控的感觉，所以我的大部分时间都是理智的。那种混乱的时间控制和情绪控制，只是理智生活的一种调剂。可惜的是，似乎是由于我对于自己的混乱没有表现出太多的惊慌和痛苦，这一次也就没有转折点般的收获。

好吧，我又要开始学习了。

当然，我又学到了一点便是，急躁是万万不可的。又是如此，因为想要尽早的完成实验报告，导致自己这一次的情绪爆发。还是因为急躁，没有办法，自己还是要学着接受时间理论，和慢慢来。

由于这两周的拖延，自己的任务皆要延后两周，好吧，我得去实实在在的调整任务了。

计划

刚刚看了很久的学校记录，最终决定要考复旦大学，也没有什么要犹豫的，直接开始吧。
