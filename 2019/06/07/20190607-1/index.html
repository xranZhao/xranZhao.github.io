灾后记

发表于 2019-06-07 更新于 2022-11-13 分类于 B-生活录 本文字数： 2.1k 阅读时长 ≈ 2 分钟

对于我而言，灾难突发的让人猝不及防。就像是一瞬间，灵魂被替换，自己不再是自己。似乎身体里面仍有一个冷静的旁观者，在一旁，看着自己疯魔，却一动不动。

即使是写下这段话的自己，仍旧清晰的感知到心里头存在一个旁观者和主导者。只不过，这一次，旁观者想要要回自己的位置，所以便会有两者的无声战争，在一个身体里面。

不过，意识总会清明，就像人不可能永远沉于低谷，也不会永远升于高峰。

放下了手机，去接水的时候，突然撇见窗外的绿。就不愿再迈开一步，只是愣愣的看着外面的一切。阳光正好，夏风正暖。这条我看了三年的路，在这一刻，又一次的散发着不一样的宁静祥和。要回忆这条路的风景，我想我以后，应该是一辈子都不会忘的。自己总是走在这条路上，看着春夏秋冬，在这条路上上演。看着落叶纷飞，看着新芽苏醒，看着一树洋溢的青春气息。看着路上的人，或孤单，或热闹，或安静。看着树间，天边，云卷云舒。看着春风，夏风，秋风，冬风，来来去去。真是美的不成样子。

我总是会有一段时间沉迷于手机，更确切的说，是沉迷于一种情绪，一种欲罢不能的情绪旋涡。那种情绪总是会让人疯魔，也许抽烟的人便是这般，不舍放，却又知道是多大的伤害。不断的拿着手机刷着一个个无趣的消息，期待着下一个消息给自己带来的惊喜。可是最终也是惶惶终日，因为热闹背后总是一场空。就像是大观园上演的悲剧，“烈火烹油，繁花似锦”最终也只剩下“白茫茫一片好大雪”。不安心，不安心，便是我沉迷背后的浮士德。

若让我选，我定选一辈子平平淡淡，细水长流，也不要一时沉迷，最终走向末路。可，不幸的便是，这暂时不是可以选择的事情，因为我还在成长，而那欲望也在成长。我要不断学着去处理这些事情，不断学着去走向细水长流。而我也知道，细水长流，给我带来的是心安，当然如果我自己不那么紧张的话。

故事的开始，应该是一场没有间隙的工作。因为没有休息，13个小时的不间断工作，导致了我最终的妥协。这样看来，要预防这种情况的再度上演，便是四字“劳逸结合”。

我仍旧在追求着最美好的未来，没有看到现在的美好。就像窗外的路，楼外的风，便可以给我带来最大的幸福。我真正想要的，也不过是这些，也不过是踏踏实实，安安心心。就像曾国藩的爷爷，最终抛弃浪荡游子的生活，回到乡间，自己耕地，最后感慨还是自家地里的东西吃的让人踏实。他追求的不过是那份踏实，那份安心罢了。

可是这个世间，越来越难让人追求到的就是一份安心。

于我而言，不过两个原因，外在的，便是诱惑，无处不在的诱惑。内在的，便是焦虑，无声无息的焦虑。这两者一直在随着时间侵入，生活和内心。而一场浮士德的幻境结束，给予我的奖赏便是那份清醒意识。我会瞬间看清自己珍惜的，自己焦虑的，自己讨厌的，自己喜欢的。不过那份清醒，也只是一瞬间的礼物，随着时间，也会渐渐麻木。所以需要一次又一次的幻境，和破灭，一次又一次的重塑和清醒，最终走向蜕变和死亡。

诱惑于我，不过是引子，是一场好戏开头必要的报幕。于平时，依我理智的性子，我皆是警惕警惕加清醒。最让我难以看清的，其实是焦虑，看似无害，实则将我拖向深渊的罪魁祸首。我每天皆是踏踏实实，一步一步，可是仍旧克服不了潜藏的焦虑，会为了一份作业焦虑，会为了任务未完成焦虑，会为了工作时间焦虑。说到底，我现今的看书，制定规则，找到时间管理法则，都是为了避免那份焦虑。

我看不见，不代表那份焦虑不存在，也不代表自己可以免于焦虑。他们只是在暗处，悄悄的潜伏，堆积，等着那份诱惑，然后一击必中，将我拖入深渊。其实那样的放纵，那样的放任自己玩手机，不过是为了消耗那份焦虑。消耗完了，我便又是我了。

然后便是下一次的轮回，不断的轮回，累积，消耗，累积，消耗。

但是，这一次，我不想这样下去了。因为太过于懦弱，也太过于愚蠢。

那我要如何摆脱？我不知道。

我目前依旧认为，焦虑是无法消除，他就是会出现，只要我还活着。我想的是，正视焦虑，减少焦虑，延长细水长流的时间。正视，便是不逃避，不逃避便是处理。譬如我讨厌实验报告的束缚，讨厌期末考试的束缚，讨厌考研的繁琐小事，这些未到的恐惧时时刻刻都在滋生焦虑，可是我就是装作视而不见，以为视而不见，便可以真的不存在。我想这样的事情，如果我不正视，便会一直存在，然后焦虑便会一直存在。正视即处理，处理的原则之一，便是细水长流。时间是细水长流的唯一受众。也就是说，我要在预计时间内，做出最符合原则的处理。

所有的实验报告，下一周都会结束，最迟星期五，最早星期四，我还有六天时间。每天 2h，预计12h，4门课，7份实验报告。在 20190628，有4门课要结业考试。复习要开始了。还有两门要在 20190709 结束。列出来，列出那些让自己恐惧的事情，然后安排时间，在焦虑的时候告诉自己踏踏实实就好。

除此之外，还有嫉妒，嫉妒也能滋生焦虑，而且是毁灭性的焦虑，自己会什么都忘记，理智便会抛之脑后，没有办法，我唯有警惕警惕加清醒。

人生本来就没有未来的幸福，有的只有一步步踏踏实实，勇敢面对，有长远的计划，减少焦虑堆积量，延长焦虑最终爆发的时间，慢慢的，便会走向心灵自由。每天看看云朵，闻闻花香，走走大道，听听音乐，踏踏实实，便是最大的幸福。
