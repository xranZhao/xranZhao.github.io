    灵魂选择自己的伴侣。

    伊迪特·伊蕾内·索德格朗

    身份：女诗人、现代主义作家。

    资历：会德语、法语、俄语。

    观点：女权主义

    作品：《诗》《九月的竖琴》《玫瑰祭坛》《未来的阴影》《不存在的国土》

    核心观点

    • 保护内在动机的三种方法：可供性（能否提供新的可能性），演绎法（借助体系、模型和框架获得自我表达），反常识（突破既有认知边界）

    • 人类学习三大隐喻：

      获得：学习是获得知识

      参与：学习是参与的过程

      创造：学习是为了创造

    • 演绎法1 演绎法2 演绎法3 寻找元知识的方法

    摘抄

    • 好思想与坏思想区别在于，前者侧重鲜活证据，后者侧重于说服自己或他人。
    • 最核心的源头知识来自少数研究者的贡献。
    • 你是对是错，并不在于别人是否认同你。你之所以正确，那是因为你依据的事实正确无误，你的推理正确无误。
    • 每一个时代的常识，都意味着这个时代的认知边界。

    行动

    • 学习谈判技巧

      博弈论

      社会冲突

      谈判领域专业书籍

      学习专业知识，总结判断及推断，然后查看判断正确的概率。【一个实用话题最好的书往往不在该领域】

      获取【元知识】的方法之一：谷歌学术搜索【谈判】，看看引用最多的学者，找到他的通俗读物。【依据】【一手知识点是有限的】
