23年-给0岁的张小超

发表于 2023-02-05 分类于 D-信 本文字数： 456 阅读时长 ≈ 1 分钟

张小超，你好！

我是在什么样的情景下写这封信

这是一个什么项目，时限多少，我会在什么时候给你

我的预期是，在你10岁生日的时候，一起交付你，如果我忍得住的话。如果你10岁以后想要和我通信，而不是单方面听我唠唠叨叨，那我会非常开心能收到你的回复。

那个时候我35岁，你10岁，我们可以一起开启一个新的项目。

两个主题

这个世界对于你的到来满怀期待

家庭成员：

我们为此准备了什么

你要做什么，一定要相信，你是值得的，你值得来到这个世界，你值得我们一切付出和一切爱，并且你会越来越出色，你会超越我们所有人，成为非常亮眼的存在

小家庭的故事讲完了，我来讲讲你来到大世界的故事

在你来到这个世界之前，这是一个什么样的世界

女性和男性，教育和学习，工作和热爱，个人和世界

主旨：这是你25岁的姑姑对于这个世界尚不成熟的看法，这绝不是标准答案。世界是什么样的，任何人说的都不对，唯有你自己去认识去体验去受苦，那才是你的世界，我期待到那时，你可以告诉我更有趣的观点，
