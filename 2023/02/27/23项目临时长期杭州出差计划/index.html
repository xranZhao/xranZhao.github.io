    23项目临时长期杭州出差计划

    发表于 2023-02-27 更新于 2023-03-02 分类于 C-试用期项目 本文字数： 4.9k 阅读时长 ≈ 4 分钟

    目前遇到最大的谈判事件

    详细话术

    目前已得知杭州6个月长期出差补助要求：

    2500元/月，不包往返差旅，不包住宿，无其他出差补助

    本人一直非常信任公司，并时刻准备好承担工作范围内必要的工作任务。在如今百草味项目紧缺人力，且无法通过借调杭州同事的情况下，我时刻告诫自己调整心态，在团队和公司需要我的时候能承担责任。

    但确实因现实因素所限（家中独子父母身体不好，每月要陪护在汉复查，也就是出差半年内需自己承担额外去汉返杭差旅费，生活用品重新置办寄送，并需提前寻找短租房，同时承担杭州更高消费水平…），现有2500每月出差补助实难支撑我在杭州生活。

    我申请每月3500出差补助，按月补助到卡，出差期间每月一次往返武汉高铁票，实报实销。

    百草味项目在3月即将启动，考虑项目时间紧张，我也需要一定时间找到合适短租房。在此诚心恳求各位领导能考虑我的申请，重新商讨出差补助条件。

    下面是我针对补助所写明细，

    对以上各别事项进行详细解释：

    1. 日常生活水平支出基于官方统计局-武汉市和杭州近1年人均支出水平倍率计算，武汉年均支出： 杭州年均支出：，相差倍率为1.3，现本人在武汉月均支出保持在2000+（附件附上自己的记账数据及支付宝流水），基于倍率计算杭州消费水平提升2000*0.3=600/月
    2. 额外生活用品指：在武汉生活已具备，但到杭州以后需重新置办的生活必需品
    3. 寄送行李费用指：需要把睡觉棉絮+6个月所需衣物寄送杭州出租屋，按照顺丰寄大件标准，1次往返价格总计
    4. 往返武汉差旅：因母亲手骨骨折，需往返武汉陪同治疗，附件是近期医疗账单和诊断报告（若后期母亲病情稳定，不需要我返汉探望，我也会尽量不往返武汉，为公司节省经费）

    再此再次恳切请求各位领导，能够在现有出差补助上再给予我多一点支持，我定会将领导帮助铭记于心。不负各位领导期望，努力工作，为公司争取更大利益。

    上级领导话术：

    娜姐娜姐，我按你说的梳理好了，我是直接拉 龚月，宗介，神乐和你，在群聊里说诉求嘛？还是我先提前和神乐打个招呼，获得她的同意再拉她？

    流程梳理

    • 自己梳理处理方法，个人诉求，理清谈判可能性和结构。

      见上面《详细话术》

    • 将事情原委，现有处理方法进行梳理。请教亲友，或有偿咨询，梳理细节

      事情背景是：

      杭州总公司有一个项目，我是主力且没办法远程办公（因为数据权限问题），杭州没办法找到一个人替我，现在要求我去杭州出差6个月。

      出差补助：2500元/月，不包往返差旅，不包住宿，无其他出差补助

      详细再解释一下就是，我得自己去杭州找短租房自己付押金出水电，出杭州返武汉之间所有交通费我自己出，且没有其他出差补助。公司只每个月出2500作为补助。

      我不能接受这个条件，已和HR1（和我沟通，平时关系还可以，她也觉得有点离谱）表明我的意思，HR1和她的直属上级HR2（性别男，已多次无脑开除公司技术员工，且经常压价员工出差费或者提薪要求）沟通，HR2明确表示无调整可能。

      我同步自己的想法和我的直属上级，直属上级要求我梳理清楚自己的诉求，然后拉她和其他领导一起商讨，她嘴上说帮我申请，但鉴于其他同事类似经历，已判断出她在出差补助调整上没有话语权，预估她的上级（即我的上上级）有话语权。

      3. 今天我在休息，明天周二我想直接拉上级+上上级+HR1+HR2建一个群聊小组，然后把我截图的这段话术发到群里，表明我的诉求。这样流程是否合规？需不需要先提前和我的上级和上上级发一句话，告知事情背景+征得请求？

        举例，“神乐老师您好（上上级），因为百草味项目需要我来杭出差6个月，HR出具了出差补助要求，因为现实条件所限，我对此他们提出的条件实战有点难接受，很希望能得到领导您的帮助，稍后可以邀请您入群一起商讨一下嘛”

        如果需要发的话，上面这个话术可以嘛？需要和HR1，HR2先单独说嘛，还是在获得上级和上上级同意以后，直接拉他们就行？

      4. 如果HR在沟通以后仍然不同意，我的最低要求是 3000元每月+开始去杭州，和最后返回武汉的交通补助报销。如果他们这个条件都不能答应的话，我准备和公司硬抗了，我死活不会去出差，即使项目已经开始，到时候最坏结果就是公司主动因为我不同意工作安排把我开除，我要争取到n+1劳动赔偿。我的问题是，因为公司出差补助不合理，而不同意公司工作安排，若被开除，我是否可以得到劳动赔偿？如果想要得到劳动赔偿，需要留存什么样的证据？

    • 真实实践推进，理清最坏结果，为最坏结果做准备。

    一次询问律师意见

    我拒绝出差，公司开除我，我能否获得赔偿。

    再次询问律师意见

    1. 背景是：我在武汉分公司，杭州总公司有一个项目，要求我去杭州出差6个月。因为公司补助没办法担负我在杭州的日常开销，我尝试和公司沟通提高出差补助，经过了2轮谈判，公司提供的补助依旧没有办法维持我的日常开销。
    2. 我选择拒绝出差，并邮件发送拒绝理由，公司口头警告我2次，最终给我发辞退通知。
    3. 公司以“经济性裁员”“严重违反用人单位的规章制度”“严重失职给公司造成了直接经济损失”“经培训和工作岗位调整以后仍然不能胜任工作岗位”为由同我解除劳务合同
    4. 就这个问题，我想以公司违约解除劳务合同为由申请劳务仲裁，有几个问题我一个一个问
      1. 我的劳务合同中有一条：“提前1个月通知，赔偿1个月工资”，是否会影响我进行劳务仲裁而拿不到双倍赔偿金？
      2. 严重违反用人单位规章制度：我如何证明自己没有违反单位规章制度，核心证据是什么。列出公司的规章制度，出差补贴制度，
      3. 严重失职给公司造成直接经济损失：公司说因为我没去而导致项目没办法推进，我能否用 公司一直没有以邮件形式发出正式通知，仅口头通知我，我也没有明确答应要去，并且我在项目即将启动前一直以积极主动态度同公司沟通我的具体困难，并列出明细，在没得到公司合适补助前提下，出具理由以邮件形式提前告知公司我拒绝出差的理由（1. 资金不足以支撑我在杭州的租房 2. 父母需要每月定期陪护检查）
      4. “经培训和工作岗位调整以后仍然不能胜任工作岗位”：没有任何违规记录，且目前已在公司工作1年3个月，并且参加今年的晋升答辩，属于不实发言
    5. 公司拉我到小黑屋讲要让我离职，但没有发任何纸质离职文件，在没有得到纸质离职文件前，我是否都可以正常上班？并且正常获取工资？
    6. 劳动仲裁是否一定要在拿到离职文件后才可以申请？在没有得到离职文件并且签署前，我是否不能劳动仲裁

    1. 签订合同时曾经同步签约竞业协议，我看法律条件中说，公司解除竞业协议，也需要赔付竞业协议3个月，2500*3的钱，是这样的吗，还是说只要公司同我签署竞业解除，我就直接接受，没有任何额外赔偿。
    2. 上面都是为了防止和公司走到最后一步的一些疑问，实际上，我现在和公司正在进行第2轮补助金的商讨，公司没有走正规的出差安排申请，我也没有走正式的同意出差申请流程，我在公司没有出具正式出差安排申请之前，是不是可以维持原样？既不写邮件拒绝，也不接受，然后一直不去杭州，这样能行吗？

    专业意见整理

    小兵哥：

    1、话术可以，“且报销我出差半年期间，去汉返杭的交通差旅费用”建议改为：“出差期间每月一次往返武汉的高铁票，实报实销”。这样标准明确。
    2、3500肯定要提，讨价还价嘛，你心理价位可以定在3000，估计有难度。或者明确调整为：“公司负责租房或公寓成本，实报实销，加60元/天的餐补和交通补助”，推翻HR2的思路；
    3、群是一定要拉的，但越级沟通前，建议先让你的直属上级知情，视他的态度，灵活决定要不要告知他你想拉群的想法。如果他也支持你寻求更多帮助，那你就跟他说拉群，看他的反应。如果你评估他不支持你拉群，而他又不能解决你的问题，那么你就最后时刻通知他一声，说你要拉群。
    如果都做了硬抗公司的准备，就不要在乎流程是否合理了，解决问题才是首要的。
    4、请两位HR专家解答。一般而言，公司都怕劳动仲裁，如果到了那一步，那就让公司明确知道你会寻求劳动仲裁的想法，公司会评估的。

    另外，你看看你的劳动合同，有没有明确写办公地点的条款，如果有就更好。

    总结

    1. 积极主动向公司表明诉求，同时向上级领导及上上级表明自己的工作态度
    2. 若HR依旧维持现有条件或者低于3000每月+交通差旅报销，拒绝出发前往杭州（但务必不能出现任何表明拒绝的词语），统一话术，不管谁问都是：我非常想要帮助公司完成这个项目，项目时间这么紧张，禁不起这么耗下去。但确实是公司差旅条件不能支持我在杭州生活，我在努力试图和公司协调，但HR好像一直不能理解这会影响多少公司的业务，你说我要怎么办呢

    感慨

    总算是完成了，在我和龚月确定出差逻辑的一瞬间，飞书弹出晋升公告，之前经历的所有的苦难，在一瞬间全部结束，就好像是上天送给我的礼物一般，刚刚好。

    在考公申论还剩30分钟时，我在纸上写下，“学会谈判+向上管理+学会勇敢”。

    问我，是真的在为这2500争取嘛？

    是也不是，人生在世，确实要计较收入，才能好好高质量的活着。但同时，这件事，精准映射出我内心的缺失和懦弱，害怕提要求，不会协商，不懂向上管理。我焦虑恐慌，害怕面对，其实不是害怕面对人，或者面对事件。而是害怕面对自己的弱点，害怕未知。所以我告诉自己，这次不管结果如何，过程多煎熬，我都会面对，一点一点迈向前方。

    最终我将这个事件完整过程记录，才发现，原来我可以克服呀，原来我在一步一步变得更好。虽然很痛，但我真的在长大。
