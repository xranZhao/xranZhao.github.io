年度项目-《未来日记》

发表于 2022-05-04 更新于 2022-10-11 分类于 Project 本文字数： 3.4k 阅读时长 ≈ 3 分钟

少年，你的未来在这里

终于在五一假期即将结束的时候开始了这个项目，《未来日记》

目前在进行的一个项目《过去的自己》，与之对应的，就是今天新开的项目。那这两个项目对于我来说，都是没有结束期限的，也许1年，也许5年，也许10年，也许就是一辈子。

冥冥之中，我总觉得自己是独一无二的，总有一些东西在等着我，我这一生注定是不平凡的。即使现在的我如此平凡，如此普通。

我今年24岁，度过了我的第二个本命年。也预料到未来的12年，对于我来说，将是极为璀璨的12年。我会找到自己的爱，幸福，自由和向往。我会抛却庸俗，抛却懒惰，抛却畏惧，去拥抱这个世界。我会去读更多的书，去认识更广阔的世界，去找到独属于我的光华。

我永远不会，也绝不会，屈服于庸俗，屈服于禁锢，屈服于社会压力。我将竭尽所能，去探索这个世界，去找到真正的自己。

下面是我的《未来日记》，他将不断更新，不断被记起。并会在未来悠悠岁月里，成为我人生前行的光
