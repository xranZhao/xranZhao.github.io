20210719-求职规划日记

昨夜辗转反侧睡不着，上网按照小哈哥的方式去找工作，在武汉，杭州，上海三座城市找运营类工作。

遇到的问题：武汉本地工作机会呈现两极化。技术类工作机会越来越多，运营类工作非常少。在非一线城市很难有公司会将运营岗位和销售岗位分割开，目前我的工作内容也更多偏向于销售而不是运营，是执行者而不是策略制定者。

生活变的非常难熬，业绩压力很大，每天都是在”赶快出单！“和”为什么不出单？“的氛围里面生存。那个地方就是战场，需要我鼓足勇气才能迈入的地方。

今天和我的同事说了我要离职的想法，同事和我沟通了30分钟。

大意如下：

全世界就你最委屈，你压力最大。你是没有经历过电销最辛苦的时候。你为什么不想在这个行业里面待了，就是压力太大吗，你现在扪心自问，压力大吗？还是你给自己找的借口？就是你自己的自制力太差了。

你要是真的累，你都没有时间去抱怨累。我们现在最迟11点都能下班，都是下午13点才来上班，字节算是8小时工作制了，没有什么强制加班的要求。

买课能抽20%，现在市面上哪个销售可以得到，你就是没经历过社会毒打

说实话，理智上我是认可他的说法的，但是感情上，我还是希望能够休息一段时间，这样的生活真的挺磨人的，每天一睁眼就是工作，周六周天加班是常态，从来没有过完整的休息时间。并且现在的业绩压力是越来越大，KPI指标是成倍的增加，工作复盘的时间也是成倍的增加。
但是我不想当销售了，也不想做销售主管，我想要找到未来更长远的规划，原来我找工作可能真的是想要逃避，想要逃离996的生活，但是现在我找工作，是想要知道我到底可不可以成为一名运营而不是一名销售，可不可以去找自己喜欢的事情。

我又通过知乎添加了一位从北上广一线城市回归的程序员，现定居武汉。通过沟通，大概我也知道我可以在哪些城市定下来，可以考虑除了武汉以外的哪些城市。

一、北上广我目前不考虑，生活成本高+勇气不足。我从一出生到现在住过最大的城市就是武汉，还是一个新一线城市，作为一个以安稳度日为目标的女孩子，从未有过去北上广闯一闯的雄心壮志。

二、武汉可以投简历，但是想要在武汉有长远发展，或者产生降维打击，还是比较难的一件事情，顶多是保命状态。

三、通过考研的方式去其他城市，这个还是纳入参考范围内，但即使是考上研究生，我还是会走向工作岗位。如果我是研究生以后再找工作，仍然不能够找到一个特别合适的工作，那我就是在白白走弯路。那我自己到底又喜欢什么专业，未来可以从事什么专业，我都不知道，没有尝试过，所以一切都是未知的。

结论：目前偏向于在武汉、杭州、成都等新一线城市找工作，生活压力不是很大，但是又相当于勇敢迈出一步，试错成本还是不那么高。

现在就准备开始写简历找工作了，目前已经确定下来了一个大的方向，运营方向。

为什么选运营方向呢？因为进入门槛最低，试错成本最低，成功转换行业的几率也是最大的。

但是运营又分为几大类，用户运营（社群运营-变现），渠道运营（广告投放-引流），产品运营（产品推广-产品），策略运营（战略）还有最后的独立版块—电商运营（引流-研发-战略-变现全流程）

目前我只会删除掉运营中我最不喜欢的一类，销售运营，其余的有机会都要尝试一下。

20210717-小哈哥职场辅导笔记

电话沟通40min

① 后期工作性质？到底选择什么样的工作？

目前工作007 早上9点-晚上12点 没有思考时间，没有生活，工资与付出不匹配

刚开始想法：找安稳工作，公务员或者老师或者编制内工作

小哈哥建议：现在工作占据生活90% 只需要调整到占据60% 晚上7，8点下班就可以，能够有双休，不用走极端。

在找工作时直接表明底牌，加班太过严重是离职原因之一，直接了解对方公司底牌，不要浪费时间在选择加班的公司上

最终结论：找能有正常休息的工作即可，面试时与HR沟通清楚离职原因

② 选择下一份工作的角度？

目前工作行业在教育行业，销售岗位

小哈哥建议：换行不换岗，换岗不换行，寻找与自己目前工作紧密链接的上下游公司

最终结论：换岗位不换行业，教育行业还是比较宽泛的，去找运营或者投放类工作

③ 如何快速了解一个行业？如何快速找到自己目前方向？

小哈哥建议：查看行业报告，招聘APP淘宝式逛法，收藏，并研究相应行业。

④ 如何确立公司是否适合？

小哈哥建议：不迷恋大厂，公司规模100人左右也可，创业公司不要去

⑤答疑

目前准备在7月底辞职，下周三左右向领导提辞职想法，还是裸辞

小哈哥建议：要想清楚裸辞以后规划是什么？

武汉机会太少，是否去大城市找工作

小哈哥建议：降维打击，如果现在年轻想要拼搏，那就去拼一把

最终结论：目前还是倾向于不裸辞的状态，那就意味着我需要在这周三或者提交辞职报告前找到下一份适合自己的工作，剩下最多的时间是15天，时间会比较紧迫。是否去大城市这个问题还在思考当中，我目前还是比较倾向于能在武汉继续待下去，能够找到一份稳定工作，然后提升自己发展副业。

如果裸辞，那后期的安排，第一是去准备考研，还没有想清楚为什么要考研，更像是希望自己能够有2年的缓冲时间。第二是休息15天左右，开始重新找工作，仍然是换行不换岗或者换岗不换行的方式。原来会担心自己裸辞以后执行力的问题，现在我会成熟很多，执行力的问题可以慢慢得到改善。

目前还是没法看清未来的职业规划路径，根据我对于自己性格的了解，会觉得自己80%的情况下更适合做一个执行者而不是管理者，是没有办法在工作上得到更大程度的提升和岗位提升。所以希望自己能找到一份安稳的工作，并且有时间去发展自己的副业，会觉得自我能力提升会比给公司卖命更重要。

自我介绍

我本科毕业于大连海事大学智能科学与技术专业，本科期间有过两份实习经验，第一份是在万邦财经教育机构进行线下面销，从大二时开始接触教育培训机构，了解面销引流+转化形式，线下运营模式。并且开设自己的独立公众号，开始接触线上运营

通过这两份工作积累的活动策划能力，社群运营能力，销售能力让我坚定了未来的职业走向，要成为教育领域的专业运营人，毕业以后进入字节跳动瓜瓜龙教育，从一线班主任老师开始做起，3个月升任主管，管理14人左右团队。

作为社群运营主管最核心的竞争力是能根据销售及节日节点做出针对性续费策略，并搭建了自己团队专属的SOP标准执行流程，我团队在全国50个团队中排名20% 1-10名，在武汉30个团队排名10%。

从大学接触教育培训机构开始，我的职业规划就是成为专业的运营人员，而教育是我认为发展前景最好的行业，目前希望把自己通过实习和工作积累的经验运用到策略运营这个岗位，希望您能给我一个机会。

为什么离职

  1. 我是社群运营出生，在社群运营中我的长项是运营策略制定及数据分析和活动复盘，一是希望自己能跳出既定舒适圈不断挑战自己，二是自己能真正发挥长处
  2. 一直的职业规划是“希望自己能成为专业的运营人”，策略运营是我最适合我职业长期规划的工作
  3. 其次是私人原因，在最近家庭我外婆离开了，促使我思考目前的职业走向和规划

策略运营

  1. 根据业务部的阶段性目标制定运营策略，配合业务部结合续费节点及时调整课程售卖策略
  2. 提供最新活动物料支持，能在最短时间响应业务部需求
  3. 对业务部业务情况进行数据分析和活动复盘，发现并解决问题，对现有策略不断优化复盘
  4. 有创新能力和敏锐度，能够及时观察市场社群运营走向，提供更有效策略进行对照参考

业务考察

目前我最大的需要提升的点：我社群运营能力是可复制的，会在一段时间内给伴鱼带来不同视角。我深感不足，关于社群运营更多的策略动作和销转动作仍然需要我不断去自我学习和成长，但是请给我一个机会，语言很单薄，但是我的执行力和成长力一定会保证持续自己学习。

故事发生在晚上17点32分，最近一段时间上班漫长的让人窒息。我在恍惚中，写下了离职的话。我的领导第一时间问我，“能不能把这期班带完？”

我很难熬，没有回复，然后被叫到办公室进行了长达1个小时的沟通，最终我还是说服他。他说要和上级沟通一下。

然后他的上级又来了，和我沟通了将近2个小时，说服了我。我太乱了，甚至我自己都不知道我自己在干什么，然后我就被说服了。

走出了那个会议室，我突然人间清醒，最后给上级的上级发了还是决定辞职的想法。

今天一天真的过的太艰难了，对我真的是降维打击，裸辞也不开心，不辞也不开心，我好像丧失了开心的能力。不过我唯一确认的一件事情是，现在的我，真的急需要离开那个地方。

不管是那个地方，有多好的工作前景，有多好的工作发展，我能得到多大的成长，我都需要离开，因为我的心要坏了。

对于自己性格的思考，不知道从什么时候开始，我觉得别人对我好，是一件特别让我愧疚的事情，我会负罪感很强，很希望能帮到别人，这种性格到底是如何形成的？

今天我的领导们一个一个的来劝我，我就越来越想疏远这个环境，我感觉我待不下去了。

谁来给我个安慰，我真的感觉辞和没辞是一样的，我还是要交接工作，还是要熬着，熬到全部结束。

在我提出离职的想法时，所有人的统一问句，“你想好下一步没有？”我没有，我现在一片混乱，没人感觉到我真的要崩溃了吗，我好希望能有个人告诉我，“没关系，你可以一直赖着。”

我没有办法从这次的裸辞事件中走出来，感觉自己完全的失败者。
